Reservoir and funnel for grit for a waterjet cutter, such as Ben Krasnow's at https://www.youtube.com/watch?v=Lg__B6Ca3jc .  Hang reservoir over funnel, clip gate onto reservoir, plug grit tube into funnel.  Expects a 3/8" OD tube (and, less stringently, 1/4" ID.)

Run get_deps.sh to pull dependencies into a linked folder in your home directory.
